# ETHDenver 2026 Canton Bounty
Total Prize Pool: $15,000

## Track 1: Best Privacy-Focused dApp Using Daml

**Prize Pool: $8,000**

Build a dApp on Canton L1 using Daml & dpm that demonstrates creativity and innovation while focusing on Canton Network's privacy-first model.

### Prize Breakdown
- **1st Place**: $5,000
- **2nd Place**: $2,000
- **3rd Place**: $1,000

### Required Submissions

**ALL SUBMISSIONS MUST INCLUDE:**

1. **Functional Deployment**: Deployed and working on Canton L1 DevNet

2. **Meaningful Daml Usage**: Smart contracts written in DAML (not wrappers around other chains)

3. **Open Source**: Public GitHub repository with accessible, reviewable code

4. **Documentation**:
   - Clear README with setup/installation instructions
   - Explanation of privacy model and data visibility per party

5. **Working Demo**:
   - 2-5 minute video demonstrating core functionality
   - Live demonstration of your project

### UI/UX Requirements

Choose one of the following:

**Option A: Web-based dApp**
- Published to a publicly accessible demo URL
- Clear separation of what different parties can see (showcasing Canton's privacy model)
- Clear indication of which "party" the user is acting as
- Demonstration of data visibility controls
- Basic responsive design (mobile-friendly preferred but not required)

**Option B: Command-line Application**
- Packaged in a Docker container OR standalone executable (Windows, Linux, or MacOS)
- Clear help documentation
- Example commands demonstrating privacy features
- Output showing which parties see which data
